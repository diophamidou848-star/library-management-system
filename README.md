# Library Management System

## Description

Library Management System is a console-based Java application connected to a PostgreSQL database.

The application allows users to:

- Manage books
- Manage members
- Borrow books
- Return books
- View borrowing records

## Technologies Used

- Java 21
- PostgreSQL
- JDBC
- GitHub

## Features

### Books

- Add Book
- View Books
- Update Book
- Delete Book

### Members

- Add Member
- View Members
- Update Member
- Delete Member

### Borrowings

- Borrow Book
- Return Book
- View Borrowings
